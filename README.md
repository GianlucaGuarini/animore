# Installation

Via npm
```shell
$ npm i animore -S
```

## Script import

Via `<script>`

```html
<script src="path/to/animore.js"></script>
```

Via ES2015 modules

```js
import animore from 'animore'
```

Via commonjs

```js
const animore = require('animore')
```
